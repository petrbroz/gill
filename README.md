# gill

Yet another path tracer. See [examples](https://github.com/petrbroz/gill/wiki/Gallery).

## Dependencies

- C++11
- cmake
- libyaml
- Doxygen (optional, used for building docs)
- ImageMagick (optional, used to embed text into images)

## Building

```
mkdir build
cd build
cmake ..
make
```

## Running

```
scripts/run-examples
```
